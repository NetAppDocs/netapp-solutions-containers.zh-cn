= 应用程序生命周期管理
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ../media/


[role="lead"]
要创建一个应用程序并跨一组集群管理它，

. 从侧边栏导航到“管理应用程序”，然后单击“创建应用程序”。提供您想要创建的应用程序的详细信息，然后单击“保存”。
+
image:redhat-openshift-078.png["创建应用程序"]

. 应用程序组件安装完成后，该应用程序就会出现在列表中。
+
image:redhat-openshift-079.png["应用程序列表"]

. 现在可以从控制台监控和管理该应用程序。

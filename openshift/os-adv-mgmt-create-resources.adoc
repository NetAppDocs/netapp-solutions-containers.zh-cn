= 在多个集群上创建资源
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ../media/


[role="lead"]
Kubernetes 的高级集群管理允许用户从控制台同时在一个或多个托管集群上创建资源。例如，如果您在不同的站点拥有 OpenShift 集群，且这些集群由不同的NetApp ONTAP集群支持，并且想要在两个站点配置 PVC，则可以单击顶部栏上的 (+) 号。然后选择要创建 PVC 的集群，粘贴资源 YAML，然后单击创建。

= 可观察性
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ../media/


[role="lead"]
Kubernetes 的高级集群管理提供了一种监控所有集群中的节点、pod、应用程序和工作负载的方法。

. 导航至观察环境 > 概览。
+
image:redhat-openshift-082.png["可观察性主页"]

. 所有集群中的所有 pod 和工作负载都根据各种过滤器进行监控和排序。点击Pods即可查看相应数据。
+
image:redhat-openshift-083.png["观察 Pod"]

. 基于各种数据点对集群中的所有节点进行监控和分析。单击节点可以深入了解相应的详细信息。
+
image:redhat-openshift-084.png["观察节点"]

. 所有集群都根据不同的集群资源和参数进行监控和组织。单击“集群”可查看集群详细信息。
+
image:redhat-openshift-085.png["观察集群"]
